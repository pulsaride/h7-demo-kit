# H7 Demo Kit Reference

Public, self-contained documentation for evaluating Pulsaride H7 (kernel-level LLM Agent Hijack detection).

## What this reference site gives you

- A single navigation point for Tracks A/B/C.
- Operational commands for local validation.
- Security, compliance, and governance references in one place.
- A concise map of key files and APIs.

## Audience

- Security engineers validating detection and attestation.
- Compliance auditors collecting evidence for DORA, NIS2, and EU AI Act mapping.
- Technical evaluators who need a reproducible demo path from cold clone.
